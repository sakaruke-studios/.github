# Sakaruke Studios

Projetos abertos para aprender, criar e automatizar.

Publicamos modelos de aplicativos, scripts, ferramentas de IA, bots para Discord,
utilitários para Ventoy, sistemas e jogos. Nosso objetivo é oferecer projetos que
qualquer pessoa consiga instalar, entender e adaptar.

Veja todos os projetos no nosso [catálogo](https://github.com/sakaruke-studios/project-catalog).

## Como participar

Você pode abrir uma *issue* com uma ideia, relatar um problema ou enviar um *pull request*.
Nunca publique tokens, senhas ou outros dados pessoais nos repositórios.
